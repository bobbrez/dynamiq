# Dynamiq

A dynamic priority queue extension for [Sidekiq](https://github.com/mperham/sidekiq).

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'dynamiq'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install dynamiq
